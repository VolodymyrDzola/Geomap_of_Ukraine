# Інтерактивна карта районів України (136 районів)

Веб-додаток на базі **Leaflet.js**, призначений для інтерактивного відображення оновленого адміністративно-територіального устрою України (136 районів та межі областей).

## 🌟 Основні можливості

- **Повна сумісність з новим устроєм**: Відображаються всі 136 районів України згідно з реформою 2020 року (включаючи місто Київ та Крим).
- **Офіційна номенклатура**: Використовуються виключно офіційні назви областей (наприклад, *Одеська*, *Івано-Франківська*, *Чернівецька* замість історичних чи розмовних варіантів).
- **Оптимізована візуальна ієрархія меж**:
  - Чіткий державний кордон України.
  - Виразні міжобласні лінії.
  - Тонкі, ненав'язливі внутрішні межі районів.
- **Розумне масштабування підписів**:
  - Розмір назв областей автоматично збільшується при наближенні карти (Zoom In), щоб залишатися розбірливими.
  - Назви автоматично ховаються на дуже дрібних масштабах (Zoom < 6), щоб не створювати візуального шуму.
  - Підписи мають напівпрозоре розмиття (`text-shadow`), що забезпечує читабельність на будь-якому кольоровому фоні.
- **Вбудовані утиліти для розробника**:
  - Клік у будь-якій точці карти виводить точні координати `[широта, довгота]` у консоль браузера (F12) для зручного коригування положення підписів.

## 🛠️ Встановлення та локальний запуск

### Попередні вимоги
Переконайтеся, що на вашому комп'ютері встановлено [Node.js](https://nodejs.org/).

### Кроки запуску

1. Встановіть залежності (потрібні для роботи допоміжних скриптів очищення геоданих):
   ```bash
   npm install
   ```

2. Запустіть локальний сервер для перегляду карти:
   ```bash
   npx serve
   ```
   *Або відкрийте `map.html` через будь-який локальний сервер (Live Server у VS Code тощо).*

3. Перейдіть у браузері за адресою, яку вкаже сервер (зазвичай `http://localhost:3000` або `http://localhost:5000`).

## 🗺️ Структура геоданих та автоматична збірка

Проект містить скрипти для підготовки та очищення географічних кордонів перед візуалізацією:

- **`cleaner.js`**: Використовує бібліотеку `mapshaper` для усунення топологічних помилок (накладань, щілин) у сирих файлах геодані. Зчитує `rayony.geojson` $\rightarrow$ створює `rayony_clean.geojson`.
- **`merge.js`**: Виконує геометричний аналіз меж. Автоматично визначає приналежність кожного району до відповідної області (методом point-in-polygon на основі `regiony.geojson`), виділяє точний контур Києва та генерує:
  - `ukraine_final_map.geojson` — фінальна карта районів.
  - `precise_oblast_borders.geojson` — чисті лінії державного та міжобласних кордонів.
